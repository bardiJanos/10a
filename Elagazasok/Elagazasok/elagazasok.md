# Elágazások (eset szétválasztás)

## Elágazások típusai:
### Egyszeres egyágú

példa 1: Megvizsgáljuk, hogy a értéke pozitív-e.
```C#
int a = 12;
if (a>0)
{
    Console.WriteLine("Pozitív szám");
}
```
### Egyszeres kétágú

```C#
if (a>=0)
{
    Console.WriteLine("Pozitív szám");
}
else
{
    Console.WriteLine("Negatív szám");
}
```
